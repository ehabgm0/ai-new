## الميزات
- منصة ذكية لاقتراح الصنايعي المناسب.
- توثيق الصنايعية عبر ربط الفيسبوك.
- تقييم الصنايعية.
- تواصل مباشر عبر واتساب.
- إشعارات فورية وتجربة مستخدم حديثة.
